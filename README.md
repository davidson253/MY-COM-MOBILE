## Démarrer le projet

> **Astuce :** Pour lancer backend et frontend ensemble :
>
> ```bash
> npm i concurrently   # optionnel
> npm start
> ```

````

### Backend server :

```bash
cd backend;npm start #ou
cd backend;node server.js
npm install    # optionnel

````

### Frontend : (nouveau terminal)

```bash
cd app;npm start #ou
cd app;vite --host --port=3000
npm install    # optionnel

```
